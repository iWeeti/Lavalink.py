coroutine voice_update_handler(data)¶

This function is a coroutine.

This function intercepts websocket data from your Discord library and forwards the relevant information on to Lavalink, which is used to establish a websocket connection and send audio packets to Discord.

Example:bot.add_listener(lavalink_client.voice_update_handler, ‘on_socket_response’)
Parameters:data – The payload received from Discord.

create(guild_id: int, region: str = 'eu', node: lavalink.node.Node = None)¶

Creates a player if one doesn’t exist with the given information. If node is provided, a player will be created on that node, otherwise a player will be created on the best node in the given region (defaults to US).

Parameters:
  • guild_id – The guild_id to associate with the player
  • region – The region to use when selecting a Lavalink node
  • node – The node to put the player on
